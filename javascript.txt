 array methods
----------------
1) push : to add new items to an array(adds at last-doesnt change index values of existing variables)
2) unshift: to add new items to an array(adds at first,make changes at index value of existing variable)

3) pop: to remove items(removes items at last from the array)
4) shift: to remove first item from the array
5) sort():  to arrange an array in ascending or decesebing manner -same array
6)forEach-same as that of for of
7)map(callback) -  create array with same number of items as that of parent array
8)reduce-return a single value from the array
9)filter- to create a new array from the existing array with elements satisfying certain condition
10)flat()-2d to 1d
11)find()-to get a single value from the array - return value
12) includes()- to get a single value from the array -return true or false
13)some()-return true or false -based on condition
14)splice() - to add or remove items from a particular position in an array
15)indexOf() - return index of a particular item
